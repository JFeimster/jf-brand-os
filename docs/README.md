# Docs

This folder contains the source-of-truth documentation for Jason Feimster’s personal brand operating system.

## Subfolders
- `brand-bio/` — About pages, brand core, bios, voice, and proof points
- `resume/` — Resume and CV source files
- `media-kit/` — speaker sheet, host intros, links, and media assets inventory
- `admin/` — project rules, source-of-truth notes, thread index, and scope

## Rule
If a derivative file conflicts with a source-of-truth file, the source-of-truth file wins.
