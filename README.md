# jf-brand-os

Source-of-truth repo for Jason Feimster’s personal brand system: bios, resume assets, media kit, prompts, website starter files, and documentation.

## Purpose
This repository acts as the operating system for Jason Feimster’s personal brand across web, social, media, speaking, author pages, and future AI-assisted workflows.

## Core areas
- `docs/` — source-of-truth brand, bio, resume, and media documentation
- `prompts/` — reusable prompts for generating and updating brand assets
- `site/` — static starter website files
- `assets/` — shared images and other static assets
- `archive/` — deprecated drafts and old variants

## Initial goals
1. Maintain a clean source of truth for bios and resume materials
2. Publish a basic static personal brand site
3. Store prompts and reusable documentation in one place
4. Make the repo easy to extend into GitHub Pages, Vercel, or a headless content workflow

## Suggested next moves
- Add final brand-bio source docs in `docs/brand-bio/`
- Add resume/CV source docs in `docs/resume/`
- Add headshots and media assets to `assets/images/`
- Decide whether this site will deploy via GitHub Pages or Vercel
