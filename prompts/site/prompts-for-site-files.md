# Prompts for Site Files

## Prompt: Build static personal brand site
Generate or revise the static files in `site/` for Jason Feimster’s personal brand site.

Files to support:
- index.html
- about.html
- services.html
- portfolio.html
- contact.html
- 404.html
- assets/css/style.css
- assets/js/script.js

Requirements:
- keep the visual style sharp, dark, modern, and readable
- emphasize trust, strategy, and founder/operator credibility
- make the code easy to extend into GitHub Pages or Vercel
- use semantic HTML and simple vanilla JS
- avoid overengineering
